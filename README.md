### pFFA Legacy

This version is meant for those who are running ancient server versions. This version of the plugin supports all versions from version **1.7.10** all the way to **1.16.5**

[Download here](https://pafias.me/minecraft/pFFA/legacy/pFFA.jar)

#### Requirements

- **Java 8** or higher
- *(Optional) [ProtocolLib](https://www.spigotmc.org/resources/protocollib.1997/history)*
  - If you are running 1.7.10 ProtocolLib will be required for the NPCs, as armorstands did not exist in that version.
  - For 1.7.10 you can use **ProtocolLib 3.7.0**
  - For 1.8 or later you can use **ProtocolLib 4.8.0 or newer**
  - The latest ProtocolLib version(s) will not work, as they require more modern minecraft/java versions.
- *(Optional) [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/)*
  - Required if you wish to get kills/deaths/kdr placeholders for, for example, with a scoreboard plugin

### pFFA

This is the default plugin version, which **supports server version 1.17 or newer**.

[Download here](https://pafias.me/minecraft/pFFA/latest/pFFA.jar)

#### Requirements

- **Java 17** or newer
- *(Optional) [packetevents](https://www.spigotmc.org/resources/packetevents-api.80279/)*
  - Required for the following features:
    - NPCs, unless you're using Citizens
    - "sprint-glitch" fix, unless you're running 1.21.4 or newer
- *(Optional) [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/)*
  - Required if you wish to get kills/deaths/kdr placeholders for, for example, with a scoreboard plugin
